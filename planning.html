Objetivos de la página Web

General

Esta pagina Web tiene como objetivo general, ser un recurso de aprendizaje Y un objeto Virtual de Información OVI, para que el Usuario puede aprender una parte de la formacion que se necestia para crear sitios Web.

Específicos:

  • Tener un concimiento base acerca del lenguaje de diseño de sitios Web CSS
  • Identificar la propiedades básica de Css
  • Estructuar de manera adecuada las proiedades Display: Inline, block, Inline-block

Inicio:

Obojetivos: El objetivo con el cual se hizo el sitio Web, es para que el usuario pueda aprender a manejar los diferentes atributos que tiene el leguaje de diseño llamado Css, donde através de videos, conceptos, imagenes, se pondra evidenciar las temáticas esenciales de Css, para ello se necesita dedicación y entrega por parte del estudiante, donde debe fortalecer a diario sus habilidades, teniendo como base el sitio web.

Tematicas a desarrollar: La plataforma se desarrallarán varios tipos de propiedades, pero la que se usará es la propiedad de Display: inline, block, inline-block, donde nos basamos en los conceptos de aprendizaje la universidad UNAD.
